= 產生手動資料倉儲建置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在OnCommand Insight 迫使更新註釋（執行暫時性資料）到支援中心之後、您需要啟動Data倉儲建置。您可以等到下一個排定的建置、或是立即開始建置。



== 步驟

. 以系統管理員身分登入資料倉儲入口網站、網址為 `+https://hostname/dwh+`、其中 `hostname` 是OnCommand Insight 安裝了IsName Data倉儲的系統名稱。
. 在左側的導覽窗格中、按一下*排程*。
. 按一下*立即建置*。
