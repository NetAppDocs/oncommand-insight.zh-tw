= 設定服務層級附註
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「服務層級」附註用於將資產與其各自的服務層級建立關聯。

服務層級附註通常只能在使用自動分層的客戶環境中設定。在Insight資料倉儲中、優先使用階層。不過、最佳實務做法是使用服務層級來詳細說明資源配置成本與客戶成本。當資料倉儲中同時存在這兩種情況時、服務層級將取代層級。

步驟：

* 選擇*服務層級*註釋、然後按一下*編輯*圖示。
* 按一下「*+新增*」、然後將第一個服務層級的「名稱」和「說明」新增至註釋清單。
* 請針對其他服務層級執行相同的操作。
* 完成後、按一下*「Save（儲存）」*。
