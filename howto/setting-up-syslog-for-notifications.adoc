= 設定記錄的Syslog
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Insight可傳送系統記錄事件以取得容量或路徑違規及效能警示。



== 關於這項工作

請遵循下列基本步驟、在Insight中設定系統記錄通知：



== 步驟

. 按一下「*管理*>*通知*」、然後移至「*系統記錄*」區段。
. 勾選「*已啟用Syslog *」核取方塊。
. 在*伺服器*欄位中、輸入記錄伺服器的IP位址。
. 在* Facility *欄位中、選取與記錄訊息之程式類型相對應的設施層級。
. 按一下「 * 儲存 * 」。
