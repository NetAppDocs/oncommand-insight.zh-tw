= 資產頁面類型
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
資產頁面會摘要說明資產的目前狀態、並包含資產及其相關資產的其他資訊連結。

提供下列資產的資產頁面OnCommand Insight ：

* 虛擬機器
* Volume
* 內部Volume
* 實體主機
* 儲存資源池
* 儲存設備
* 資料存放區
* Hypervisor
* 應用程式
* 儲存節點
* qtree
* 磁碟
* VMDK
* 連接埠
* 交換器
* 網路
* 物件儲存（例如、Atmos、Centera、Amazon S3）
* 區域


對應與遮罩資訊可在「區域」、「Volume」、「VM」和「主機/ Hypervisor」資產頁面的表格中檢視。

[NOTE]
====
摘要資訊可用於物件儲存資產、不過您只能從「資料來源詳細資料」頁面存取此資訊。

====
