= 在一種資料來源上安裝修補程式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
上傳資料來源修補程式之後、您可以將其安裝在相同類型的所有資料來源上。



== 開始之前

您必須已上傳要安裝在某種資料來源上的修補檔案。



== 步驟

. 在Insight工具列上、按一下*管理*。
. 按一下*修補程式*。
. 從「動作」按鈕選取*「套用修補程式」*。
. 在「*套用資料來源修補程式*」對話方塊中、按一下「*瀏覽*」以尋找上傳的修補程式檔案。
. 請查看*修補程式名稱*、*說明*和*受影響的資料來源類型*。
. 如果選取的修補程式正確、請按一下*套用修補程式*。
+
相同類型的所有資料來源都會更新此修補程式。
