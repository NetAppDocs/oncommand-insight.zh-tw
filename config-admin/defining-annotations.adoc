= 定義註釋
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
自訂OnCommand Insight 功能以追蹤資料以符合企業需求時、您可以定義所需的任何專業註釋、以便提供完整的資料圖片：例如資產生命週期結束、資料中心、建置位置、儲存層或磁碟區、 和內部Volume服務層級。



== 步驟

. 列出環境資料必須關聯的任何產業術語。
. 列出環境資料必須關聯的企業術語、但尚未使用商業實體來追蹤這些術語。
. 找出您可以使用的任何預設註釋類型。
. 找出您需要建立的自訂附註。
