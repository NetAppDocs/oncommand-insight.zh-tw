= 摘要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
資產頁面上的「摘要」區段會顯示特定資產的相關資訊摘要、並顯示與資產相關的問題、以紅色圓圈表示、並提供超連結、可連結至相關資產的其他資訊、以及指派給該資產的任何效能原則。
