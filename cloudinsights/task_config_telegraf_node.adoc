從安裝代理程式的節點收集度量。Cloud Insights



== 安裝

. 從 * 可服務性 > 收集器 * 中、選擇作業系統 / 平台。請注意、安裝任何整合資料收集器（Kubernetes、Docker、Apache等）也會設定節點資料收集。
. 依照指示設定代理程式。這些指示會因您用來收集資料的作業系統或平台類型而有所不同。
